# 19 — JD Skills

## Purpose

Catalogue of skills referenced by Contacts (consultants), Job Descriptions, and Prospect "Services Discussed". May be AI-curated.

## Primary entity: `JDSkill`

| Field | Type | Required | Notes |
|---|---|---|---|
| `name` | String | yes | unique, case-insensitive |
| `description` | String (multi) | no | |
| `definedByAI` | Boolean | yes | default false; true when generated by an AI suggestion run |

## Validation rules

- `name` required, unique.
- Trim whitespace; reject empty after trim.

## Business rules

- AI-defined skills are flagged so a human reviewer can promote or merge them.
- Renaming a skill cascades to all `Contact.skillset` arrays referencing the old name (production: store as lookup-by-id to avoid this; the prototype stores names).
- Cannot delete a skill that is referenced.

## Filters

Search (name, description), Defined By AI (toggle).
